Welcome to Coverage Maximization Game

Text Instructions:

1. Players in this game are represented by tokens
2. You control the token with a lock on it

3. Each player's agent cover tiles in the surrounding (indicated by green tiles), except obstacles (black tiles)

4. If multiple agents are near a tile, an overlap occurs as indicated by dark green tiles

5. The objective of this game is for every player to cooperatively cover as much space as possible
6. This is a time-based game, and you may move your agent every 2 seconds
7. The first round is a tutorial and your actions will not be recorded
